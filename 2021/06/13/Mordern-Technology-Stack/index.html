Apache Jmeter

  2. Support Http/https, JMS, SMTP, Soap & REST Service
  3. Provide an option to record our performance scenario
  4. We can write code in BeanShell but also support other languages like groovy, java etc

  6. Easy Continuous Integration through 3rd party Open Source libraries for Maven, Gradle and Jenkins

Redis

It’s an open source tool that runs as a service in the background that allows you to store data in memory for high-performance data retrieval and storage.

VUE.js

Vue (pronounced /vjuː/, like view) is a progressive framework for building user interfaces. Unlike other monolithic frameworks, Vue is designed from the ground up to be incrementally adoptable. The core library is focused on the view layer only, and is easy to pick up and integrate with other libraries or existing projects. On the other hand, Vue is also perfectly capable of powering sophisticated Single-Page Applications when used in combination with modern tooling and supporting libraries.

Big Data management tool, a powerful open-source software platform.

It is used for managing clusters fo reliable storage and to analyses instructed Data

  1. Large Data, generally larger than 10 TB

Apache Flink

Apache Flink is an open-source stream processing framework for distributed, high-performing, always available and data streaming applications.

It also provide supports for batch processing, Graph processing, Iterative processing.

Flink is the next generation Engine for stream processing. It can considered as 4G for Big data processing.

Speed wise: Flink > Spark > Hadoop

Flink give use low latency and high throughput application

Robust Fault-tolerance, application can restart exactly from exactly from same point where it failed

Has rich set of libraries - For Graph processing, Machine Learning, String handing, Relational Apis etc.

Flink’s application state is re-scalable, possible to add resources while app is running. Also maintains exactly once semantics.

LDAP Active Directory

It is an application protocol used over an IP network to manage and access the distributed directory information service. The primary purpose of a directory service is to provide a systematic set of records, usually organized in a hierarchical structure. It’s similar to a telephone directory that contains a list of subscribers with their contact number and address.
